# Customer Churn Prediction

A machine learning project to predict customer churn using the Telco Customer Churn dataset. This project includes both model development in Jupyter Notebook and an interactive web application built with Streamlit.

## Project Overview

This project aims to predict which customers are likely to churn (leave) using machine learning algorithms. The model analyzes customer demographics, service usage, and account information to identify churn patterns and provide actionable insights.
